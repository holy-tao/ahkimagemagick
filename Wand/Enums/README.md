# Enums

Enumeration values are pulled from ImageMagick's headers. ImageMagick does add new values from time-to-time, so it may
be necessary to update these. For this reason you should prefer using the enum values over hard-coded constants in
your AHK code as well.

Check here for values:
- https://github.com/ImageMagick/ImageMagick/tree/main/MagickCore
- https://github.com/ImageMagick/ImageMagick/tree/main/MagickWand
